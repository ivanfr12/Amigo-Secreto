Amigo Secreto

Este proyecto es una aplicación web interactiva para sortear un amigo secreto. Permite a los usuarios ingresar nombres, ver la lista de amigos, realizar el sorteo y resetear el resultado para volver a jugar.

Funcionalidades

Agregar amigos: Permite ingresar nombres en un campo de entrada y añadirlos a una lista.

Validación de entrada: Se asegura de que el nombre no esté vacío antes de agregarlo.

Visualización de la lista: Muestra los amigos agregados en una lista en la interfaz.

Sorteo de amigo secreto: Selecciona un nombre aleatorio de la lista de amigos y lo muestra en pantalla.

Resetear sorteo: Permite limpiar el resultado para realizar un nuevo sorteo.

Archivos principales

index.html: Contiene la estructura de la página, incluyendo el formulario de entrada, la lista de amigos y los botones de acción.

app.js: Archivo JavaScript que maneja la lógica del juego, incluyendo captura de datos, validaciones y actualización de la interfaz.

Uso

Ingrese un nombre en el campo de entrada y presione "Añadir".

Se mostrará la lista de amigos agregados.

Presione el botón "Sortear amigo" para seleccionar un amigo secreto aleatorio.

Si desea reiniciar el sorteo, presione el botón "Resetear sorteo".

Tecnologías utilizadas

HTML5: Para la estructura de la página.

CSS3: Para el diseño y estilos.

JavaScript (ES6): Para la lógica y manipulación del DOM.

Mejoras futuras

Implementación de estilos CSS más avanzados.

Posibilidad de eliminar nombres de la lista antes de sortear.

Agregar una animación para el sorteo.
